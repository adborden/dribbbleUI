adborden's Dribbble Show

This is an exercise to show some javascript work. The idea was to create a dynamic image library by pulling data from an external site (Dribbble).

Details

Load the first nine items from each list (popular, debuts, everyone). Render the shots dynamically when the list's tab is clicked. Reloading the page should not be necessary. Add to the list each time the user clicks "Load More" or as the user scrolls.

Featuring

  • Dribbble API
  • Skeleton grid
  • jQuery
  • Backbone.js
  • Underscore.js
  • Infinite scroll
  • Responsive UI (try resizing)
  • and more...
